# utils

## flow

Performs left-to-right function composition. The first argument may have any arity, the remaining arguments must be unary.

**Signature**

```ts

  ab: (...a: A) => B,
  bc: (b: B) => C
): (...a: A) => C

  bc: (b: B) => C,
```
