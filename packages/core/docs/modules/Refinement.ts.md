# lifting

## liftOption

Returns a `Refinement` from a `Option` returning function.
This function ensures that a `Refinement` definition is type-safe.

**Signature**

```ts
export declare const liftOption: <A, B extends A>(f: (a: A) => any) => Refinement<A, B>
```

Added in v3.0.0
